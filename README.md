# -2. 프로젝트 주요내용
2.1 배경 및 필요성

최근 여행 수요가 증가하면서 개인 맞춤형 여행 일정에 대한 관심이 높아지고 있다. 하지만 여행 일정을 계획하기 위해서는 관광지 정보, 이동 시간, 맛집, 숙소 등을 여러 사이트에서 찾아보고 비교해야 하므로 많은 시간과 노력이 필요하다. 특히 해외여행이나 처음 방문하는 지역의 경우 여행 동선, 관광지 운영 시간, 식사 장소, 이동 거리 등을 고려하여 효율적인 일정을 계획하는 것이 쉽지 않다.

기존 여행 관련 서비스들은 관광지 정보 제공이나 숙소 예약 기능 중심으로 구성된 경우가 많으며, 사용자의 여행 스타일과 일정 조건을 종합적으로 고려하여 여행 계획을 자동으로 생성해주는 기능은 부족한 실정이다. 또한 동일한 지역을 여행하더라도 사용자마다 여행 목적이 다르기 때문에 “맛집 위주의 여행”, “관광 명소 중심 여행”, “가성비 여행”, “휴식 중심 여행” 등 다양한 요구를 반영할 수 있는 서비스가 필요하다.

이에 본 프로젝트에서는 사용자가 여행지, 여행 기간, 예산, 선호 테마 등의 정보를 입력하면 이를 분석하여 개인 맞춤형 여행 일정을 자동으로 생성해주는 AI 기반 여행 일정 자동 생성 서비스를 개발하고자 한다. 본 서비스는 여행 준비 과정에서 발생하는 시간과 정보 탐색의 부담을 줄이고, 사용자에게 효율적인 여행 계획을 제공하는 것을 목표로 한다.

또한 본 프로젝트는 인공지능 기술, 웹 서비스 개발, 데이터 처리 기술을 종합적으로 활용하는 프로젝트로서 소프트웨어 전공 학생의 실무 역량을 향상시키고 캡스톤디자인 교육의 목적에 부합하는 과제로 판단된다.

2.2 주요 목표

본 프로젝트의 주요 목표는 사용자 입력 정보를 기반으로 여행 일정을 자동으로 생성하는 AI 기반 여행 일정 추천 웹 서비스를 구현하는 것이다.

사용자는 여행지, 여행 기간, 예산, 선호 여행 스타일 등을 입력하면 시스템이 이를 분석하여 일자별 관광지, 맛집, 이동 순서 등을 포함한 여행 일정을 자동으로 생성하도록 한다.

세부 목표는 다음과 같다.

첫째, 사용자가 여행 정보를 쉽게 입력할 수 있도록 웹 기반 사용자 인터페이스를 구현한다.
둘째, 관광지, 음식점, 숙소 등의 여행 관련 데이터를 수집하여 일정 생성에 필요한 데이터베이스를 구축한다.
셋째, 사용자 조건을 분석하여 여행지를 추천하고 일자별 일정을 구성하는 추천 알고리즘을 개발한다.
넷째, 생성된 여행 일정을 사용자에게 직관적으로 제공할 수 있도록 웹 서비스를 구현한다.
다섯째, 실제 사용 시나리오를 기반으로 시스템의 기능을 테스트하고 서비스 활용 가능성을 검증한다.

최종적으로 본 프로젝트는 사용자가 간단한 정보 입력만으로 효율적인 여행 계획을 수립할 수 있는 서비스를 제공하는 것을 목표로 한다.

2.3 관련연구 및 기술동향

최근 인공지능 기술의 발전과 함께 개인 맞춤형 추천 서비스에 대한 연구가 활발히 이루어지고 있다. 특히 자연어 처리 기술과 추천 시스템 기술을 활용하여 사용자의 선호도를 분석하고 맞춤형 정보를 제공하는 서비스가 다양한 분야에서 활용되고 있다.

여행 분야에서도 인공지능을 활용한 관광지 추천, 여행 리뷰 분석, 일정 추천 서비스 등이 등장하고 있으며, 사용자 데이터를 기반으로 개인화된 여행 정보를 제공하는 서비스가 증가하고 있다. 또한 지도 API와 위치 기반 서비스 기술을 활용하여 관광지 간 이동 거리와 시간을 고려한 여행 계획을 제공하는 연구도 활발히 진행되고 있다.

본 프로젝트에서는 이러한 기술 동향을 반영하여 추천 시스템 기반 여행지 추천 기능과 웹 기반 서비스 기술을 결합한 여행 일정 자동 생성 시스템을 구현하고자 한다. 이를 통해 사용자의 요구를 반영한 효율적인 여행 일정을 생성하고 사용자 편의성을 높일 수 있을 것으로 기대된다.

2.4 주요 추진내용

본 프로젝트는 여행 일정 자동 생성 서비스를 구현하기 위해 다음과 같은 단계로 추진한다.

첫째, 프로젝트 요구사항을 분석하고 시스템 구조를 설계한다. 사용자가 입력할 수 있는 여행 정보 항목을 정의하고 서비스의 전체 기능과 화면 구조를 설계한다.

둘째, 여행 관련 데이터를 수집한다. 관광지, 음식점, 숙소 등 여행 일정 구성에 필요한 데이터를 수집하고 이를 데이터베이스 형태로 정리하여 시스템에서 활용할 수 있도록 구축한다.

셋째, 여행 일정 생성 알고리즘을 개발한다. 사용자 입력 조건을 분석하여 적합한 관광지를 추천하고 여행 기간에 맞게 일정을 자동으로 구성하는 추천 로직을 구현한다.

넷째, 웹 기반 사용자 인터페이스를 개발한다. 사용자가 여행 정보를 입력하고 생성된 여행 일정을 확인할 수 있도록 웹 서비스를 구현한다.

다섯째, 시스템 테스트 및 성능 검증을 수행한다. 다양한 사용자 시나리오를 적용하여 시스템 기능을 검증하고 서비스의 안정성과 활용 가능성을 평가한다.

2.5 기대효과

본 프로젝트를 통해 사용자는 복잡한 여행 계획 과정을 거치지 않고 간단한 정보 입력만으로 개인 맞춤형 여행 일정을 생성할 수 있다. 이를 통해 여행 준비에 필요한 시간과 노력을 줄일 수 있으며 보다 효율적인 여행 계획을 수립할 수 있다.

또한 사용자 선호도와 여행 목적을 반영한 맞춤형 일정 제공이 가능하여 여행 서비스의 편의성과 만족도를 향상시킬 수 있다.

교육적 측면에서는 인공지능 기술, 데이터 처리, 웹 서비스 개발 등 다양한 소프트웨어 기술을 실제 프로젝트에 적용함으로써 전공 지식을 실무적으로 활용하는 경험을 얻을 수 있으며, 캡스톤디자인 프로젝트를 통해 소프트웨어 개발 역량과 문제 해결 능력을 향상시키는 효과가 기대된다.
